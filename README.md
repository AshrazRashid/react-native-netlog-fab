# react-native-netlog-fab

It is an wrapper for react-native-network-logger

## Installation

```sh
npm install react-native-netlog-fab
```

## Usage

```js
import { NetworkLoggerFAB } from 'react-native-netlog-fab';

// In your component:
<NetworkLoggerFAB />;
```
